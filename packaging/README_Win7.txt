Axi Factory Workstation Win7 package

Target: Windows 7 SP1 x64 industrial PC, UART factory workflow.
BLE on Windows 7 is not a validation target because Windows 7 lacks the modern Windows BLE API used by Bleak/WinRT.
Run: Axi Factory Workstation.exe
Config/token files live beside the exe. Test records are stored under factory_records.
